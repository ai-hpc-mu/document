The MAI HPC cluster is managed by ICT network and system team (Faculty of Information and Communication Technology). This documentation is maintained by ICT team and the user community. How-To: Contribute to this Document

The document, user guide and source code is managed using Git and is hosted on GitHub. The recommended way for new contributors to submit information to MAI document is to fork this repository and submit a pull request after committing changes to their fork. The pull request will then need to be approved by one of the core members before it is merged into the main repository. If you feel uncomfortable or uncertain about an issue or your changes, feel free to email the mai@mahidol.ac.th.

These are the basic steps needed to start developing on Sphinx.

  1. Install Sphinx on Conda Environment

$ conda create --name workshop

$  conda activate workshop

$  conda install sphinx

$ pip install sphinx-rtd-theme

Verify version:

$ sphinx-build --version

Clone github project

$ git clone https://github.com/ai-hpc-mu/document.git

$ cd document/doc

Setup project and theme

$ vi conf.py
 html_theme = 'sphinx_rtd_theme'

  1. Create an account on GitHub.

  2. Fork the repository and add your changes (more details:)

  3. Learn how to use Sphinx

  4. Add a pull request
